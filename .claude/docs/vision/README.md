# Vision — Programming Learning Workspace

このディレクトリは「**なぜこの学習設計を選ぶのか**」「**学習者として何を目指すのか**」の根底の指針を置く場所。

## CLAUDE.md との違い

- `CLAUDE.md`: 現状のディレクトリ構成・運用規約・ジャンル一覧の SSOT（**何が / どこに**）
- `vision/`: 設計判断の根拠 / 哲学 / 学術的支えがある原則（**なぜそうするか**）

ADR（Architectural Decision Record）形式は使わない。「時点の判断を凍結する」のではなく、「現在から未来に向けた指針」として継続更新する。却下した代替案も「なぜ採用しなかったか」と一緒に残す。

## 構成

| ファイル                 | 内容                                                             |
| ------------------------ | ---------------------------------------------------------------- |
| `core.md`                | 学習の目的 / 哲学 / 学習者像 / Non-Goals                         |
| `learning-principles.md` | 学習科学 12 原則（一次研究・メタ分析の出典付き、教材設計の根拠） |
| `<topic>.md`             | 個別の探索的問い・未確定の方針                                   |
