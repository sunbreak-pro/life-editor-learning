# Code Explanation

Claude が学習中に「この概念がわからない」と聞かれた時に書き起こす解説の置き場。

## 運用

- 学習者が「`useState` の更新関数って結局何してるの？」と聞いた → Claude がここに `usestate-updater.md` を作成
- 既存ファイルがあれば追記、なければ新規作成
- ファイル名は `<topic-slug>.md`（kebab-case）

## learning-log との違い

| 観点   | learning-log         | code-explanation |
| ------ | -------------------- | ---------------- |
| 書き手 | **学習者**           | **Claude**       |
| 内容   | 自分の言葉での理解   | 概念の正式解説   |
| 目的   | 理解度の自己チェック | リファレンス     |

両方を行き来することで「Claude の解説 → 自分の言葉に翻訳」の流れができる。
