# Learning Log

セッション毎に「今日学んだこと」を `YYYY-MM-DD-<topic>.md` で残す。

## フォーマット

```markdown
# YYYY-MM-DD — <topic タイトル>

## 詰まったポイント

何を勘違いしてた？

## 解決のヒント

Claude からどんなヒントが効いた？

## 自分の言葉での説明

ググらず・コピペせず、自分の言葉で書く。読み返した時の自分が「これでわかる」と思えるレベルで。

## 次に試したいこと

派生疑問。
```

## 重要ルール

- **コピペ禁止**: ググったページや Claude の回答をそのまま貼らない。自分でタイプし直す
- **書けないなら理解してない**: 「自分の言葉での説明」が書けない場合、まだ完了マークを付けない（学習ロードマップ md / アプリ両方とも）
- **後で読み返した時の自分が読者**: 今の自分が一番わかってない、3 ヶ月後の自分が読むつもりで書く
